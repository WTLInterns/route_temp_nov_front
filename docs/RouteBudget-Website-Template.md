- Log entries via mobile camera
- Instant insights on the go

- Real-time GPS tracking via device feed with map visualization

## FAQs (Product)

- How do we onboard drivers/cabs?
  - Add via DriverDetails/CabDetails and assign using AssignCab.
- Do we support live tracking?
  - Yes, GPSTracking/CabInfo with WebSocket updates.
- How are expenses managed?
  - Log in Expensive; charts on Dashboard show trends and breakdowns.
- Can we generate invoices?
  - Yes, InvoicePDF supports GST, service charge, journey types, logos, signatures.
- How is servicing/compliance handled?
  - Servicing flows with receipts; DocumentExpiry monitors renewals.
- Are subscriptions/payments supported?
  - Yes, Subscription module with Razorpay.
